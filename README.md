# CONNECTION-ZERO

## Opis projektu
Gracz wciela się w zwiadowcę wykonującego misję odnalezienia porzuconego zaopatrzenia.  
Podczas eksploracji mapy należy unikać przeciwników oraz utrzymać połączenie z bazą.

Projekt zawiera:
- losowe wydarzenia,
- przeciwników poruszających się po mapie,
- system połączenia z bazą,
- poziomy trudności,
- zbieranie zaopatrzenia,
- raport końcowy z przebiegu misji.

---

# Jak pobrać projekt

## Pobranie ZIP
1. Wejdź na stronę repozytorium GitHub.
2. Kliknij zielony przycisk `Code`.
3. Wybierz `Download ZIP`.
4. Wypakuj pobrany folder.

# Jak uruchomić program

## 1. Otwórz folder projektu
Przejdź do folderu zawierającego plik `main.py`.

## 2. Uruchom terminal w folderze projektu

## 3. Wpisz komendę

```bash
python main.py
```

Jeżeli komenda `python` nie działa, użyj:

```bash
python3 main.py
```

# Wymagania
- Python 3.x

Program korzysta wyłącznie z bibliotek standardowych:
- turtle
- math
- random
- os
- platform

Nie trzeba instalować dodatkowych bibliotek.
